# kdeloach.github.io

## Install

```sh
yarn install
```

## Scripts

To rebuild HTML content:

```sh
./scripts/build.sh
```

To start web server:

```sh
./scripts/serve.sh
```

To bundle JS for a single project:

```sh
cd platecalc
../scripts/bundle.sh
```

To run the dev server to watch for changes:

```sh
../scripts/watch.sh
```
